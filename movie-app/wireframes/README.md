- **Color Scheme:**
  - Background: `#0f0f1e` (Dark Navy)
  - Card: `#1a1a2e` (Darker Navy)
  - Primary: `#6c5ce7` (Purple)
  - Accent: `#ff6b6b` (Red - Favorites)
  - Text: `#fff` (White)

- **Typography:**
  - Headers: 14-20px, Bold
  - Body: 11-12px, Regular
  - Labels: 10-11px, Light

- **Components:**
  - Buttons: 30-50px height, rounded corners (3-6px radius)
  - Cards: Rounded corners (6-8px radius), with border
  - Grid: 4 columns (Desktop), responsive

## 📥 Cách Import vào Figma

### **Cách 1: Copy-Paste SVG Code**
1. Mở file SVG bằng text editor
2. Copy toàn bộ code
3. Figma → Create Board → Paste
4. SVG sẽ render như một hình ảnh

### **Cách 2: Import File**
1. Figma → File → Import
2. Chọn file `.svg`
3. Click "Import"
4. SVG được nhập vào board

### **Cách 3: Drag & Drop (nếu hỗ trợ)**
1. Kéo file SVG vào Figma workspace
2. Thả để import
